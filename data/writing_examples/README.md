# 일기 작성 스타일 예시 문장 관리

이 디렉토리는 각 일기 작성 스타일별 예시 문장을 관리합니다.

## 파일 구조

```
data/writing_examples/
├── emotional_literary.txt          # 감성적/문학적 스타일 예시
├── objective_third_person.txt      # 객관적 3인칭 스타일 예시
└── first_person_autobiography.txt  # 1인칭 자서전 스타일 예시
```

## 사용 방법

### 1. 예시 문장 추가

좋아하는 작가의 문장이나 마음에 드는 표현을 한 줄씩 추가하세요:

```txt
# emotional_literary.txt
긴 회의들 사이로 피로가 밀려왔다.
석양이 창문을 물들일 때쯤, 나는 하루의 무게를 느꼈다.
그날의 기억은 안개처럼 흐릿하게 남았다.
```

### 2. 주석 사용

`#`으로 시작하는 줄은 주석으로 처리되어 무시됩니다:

```txt
# 이것은 주석입니다 (무시됨)
실제 예시 문장입니다 (사용됨)
```

### 3. AI에 미치는 영향

- **EMOTIONAL_LITERARY 스타일**: 예시 문장들이 AI 프롬프트에 포함되어 해당 문체를 학습합니다
- **다른 스타일**: 현재는 기본 프롬프트만 사용 (향후 확장 가능)

## 예시: 좋아하는 작가 문체 적용하기

1. `emotional_literary.txt` 파일 열기
2. 좋아하는 작가의 문장 추가:
   ```txt
   # 무라카미 하루키 스타일
   나는 커피를 마시며 창밖을 바라보았다.
   시간은 천천히, 매우 천천히 흘러갔다.

   # 한강 작가 스타일
   그 순간, 세계는 조용히 멈춰 섰다.
   ```
3. 일기 작성 시 EMOTIONAL_LITERARY 스타일 선택
4. AI가 해당 문체를 참고하여 일기 작성

## 주의사항

- 한 줄에 하나의 예시 문장만 작성
- 빈 줄은 자동으로 무시됨
- 파일이 없어도 기본 프롬프트로 동작
- UTF-8 인코딩 사용

## 확장 가능성

나중에 다른 스타일에도 예시 문장을 추가하면 자동으로 적용됩니다!
